# FAQ & Community

Willkommen im offiziellen R&D Laboratory! Hier findest du Antworten auf die wichtigsten Fragen zu unserem System.

**Wie kann ich den Entwickler kontaktieren?**
Schreib mir bei Bugs, Ideen oder Fragen einfach direkt eine E-Mail an: rotti.dev@gmail.com

**Gibt es einen Discord-Server?**
[Hier kannst du später deinen Einladungs-Link einfügen, wenn du einen Server erstellst!]

**Hilfe, mein Spielstand ist weg!**
Um deine Daten maximal zu schützen, wird dein Spielstand (Score, Hashes, Root Tokens) lokal auf deinem Gerät verschlüsselt gespeichert. Wenn du die App deinstallierst oder den App-Cache über die Android-Einstellungen komplett löschst, wird der Spielstand unwiderruflich zerstört. 

**Ich habe eine Forschung abgeschlossen, aber es passiert nichts?**
Forschungen dauern Zeit. Schließe die App ruhig – das System forscht im Hintergrund weiter und schickt dir eine Terminal-Benachrichtigung, sobald die Integration abgeschlossen ist!
